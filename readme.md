# Django : CRUD simple   

> version 4.2.4

## rappel des commandes :   

#### création de la base de données et des tables  

```bash
python manage.py makemigrations
python manage.py migrate  
```


#### création du gestionnaire de l'administration en ligne de Django
```bash
python manage.py createsuperuser
python manage.py runserver
```  

> se rendre à l'adresse http://localhost:8000/admin pour remplir la base de données  

Se rendre ensuite à l'adresse http://localhost:8000/gens
